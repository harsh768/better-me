# THE SYSTEM

A Solo Leveling-themed life RPG (PWA). Daily quests for training, nutrition, recovery, skincare, study (OSCP / bug bounty / AI security) and work.

**Live:** https://harsh768.github.io/better-me/

## Install on iPhone
1. Open the live URL in **Safari**.
2. Tap **Share** → **Add to Home Screen** → **Add**.
3. Launch from the home screen — it runs fullscreen and offline.

All data is stored locally on your device. Use Settings → Export to back up before switching phones.
